# sakana-widget-react Contributing Guide

## Setup

(Optional) Use corepack

```
corepack enable
```

Install dependencies using [pnpm](https://pnpm.io/)

```
pnpm install
```

### Run development environment(Storybook)

```
pnpm run dev
```

### Check and fix format and lint

```
pnpm run format:fix
pnpm run lint:fix
```
